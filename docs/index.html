Z-Waterfall

一个轻量级、高性能的微信小程序瀑布流组件，支持懒加载和自动布局计算。

快速开始 GitHub

✨ 特性

🖼️ 图片懒加载

自动处理图片懒加载，优化性能

📱 上拉加载更多

支持上拉加载更多数据

🎨 灵活的样式自定义

支持自定义样式，满足不同需求

📐 智能间距计算

自动计算并优化布局间距

🔌 原生组件模式

类似 swiper 的使用方式

🔄 数据与视图解耦

更灵活的数据处理方式

📦 安装

使用 npm

npm install @waterfall-weapp/z-waterfall

使用 yarn

yarn add @waterfall-weapp/z-waterfall

🚀 快速开始

1. 引入组件

{
  "usingComponents": {
    "waterfall": "@waterfall-weapp/z-waterfall/src/components/waterfall/index"
  }
}

2. 使用组件

<waterfall 
  column="2"
  column-gap="10"
  row-gap="10"
  bindloadmore="onLoadMore"
>
  <block wx:for="{{list}}" wx:key="id">
    <view slot="item-{{index}}">
      <image 
        lazy-load
        src="{{item.image}}"
        mode="widthFix"
        style="width: 100%;"
      />
      <view class="title">{{item.title}}</view>
    </view>
  </block>
</waterfall>

📖 API

Props

属性 说明 类型 默认值
column 瀑布流列数 number 2
columnGap 列间距（单位：rpx） number 10
rowGap 行间距（单位：rpx） number 10
loadMoreThreshold 触发加载更多的距离阈值（单位：px） number 100

查看完整示例请访问 示例页面

📝 注意事项

  • 确保图片设置了 mode="widthFix" 以保证正确的宽高比
  • 建议使用 lazy-load 属性来优化图片加载性能
  • 列表项必须有唯一的 key 值
